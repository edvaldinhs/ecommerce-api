## 🧑🏻‍💻&nbsp; Getting Started

Clone the project and access the project folder

```bash
$ git clone https://github.com/edvaldinhs/ecommerce-api
$ cd ecommerce-api
```

Run the server

```bash
node server.js
```

you can add the *--watch* parameter if you want it to restart every change.

<br>
